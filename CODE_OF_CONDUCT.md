# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Our standard

Contributors should be respectful, constructive, and focused on the work. Good
faith disagreement is welcome; harassment, personal attacks, and discriminatory
language are not.

## Enforcement

Project maintainers may edit, hide, or remove comments, issues, pull requests,
or other contributions that violate this standard. Maintainers may also block
contributors from further participation when needed.

## Reporting

Report conduct concerns privately to the project maintainer through the contact
details listed on the maintainer's GitHub profile.
